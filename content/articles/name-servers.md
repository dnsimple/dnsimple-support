Name servers are the foundation of the Domain Name System (DNS), translating human-readable domain names into IP addresses and directing internet traffic to the correct destinations. At DNSimple, we provide robust name server infrastructure and management tools, including support for vanity name servers, name server sets, and seamless domain delegation, making it easy to configure and manage your domain's authoritative name servers.

- [What Are Glue Records?](/articles/what-are-glue-records/): Understand how glue records solve the circular dependency problem when name servers are subdomains of the domain they serve.
- [What Are Vanity Name Servers?](/articles/what-are-vanity-name-servers/): Discover how vanity name servers allow you to brand your DNS infrastructure with your own domain name.
